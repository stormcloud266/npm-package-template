# npm-package-template

## Dev

```bash
yarn dev
```

## Build

The `README.md` file, `package.json` file, and all files in `src/components` will be exported.

```bash
yarn build
```

## Publish

```bash
yarn build
```

```bash
npm version patch
```

```````bash
npm patch

``````bash
npm publish --access public
```````
